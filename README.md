# CAN Trace Viewer

Phase 1 scaffold: drop a DBC-JSON file, see a summary.

## Prerequisites

- Node.js 20+
- npm
- A DBC-JSON file produced by the sibling `script.js` (`node ../script.js your.dbc`)

## Run

```
npm install
npm run dev
```

## Build

```
npm run build
```

Produces a distributable under `dist/`. For an unpacked build: `npm run build:unpack`.
